# Proyecto Web

Sitio web simple desarrollado con **HTML, CSS y Bootstrap**.  
El proyecto muestra una interfaz responsive utilizando el sistema de grillas y componentes de Bootstrap.

---

## Tecnologías

- HTML5
- CSS3
- Bootstrap

---

## Estructura del proyecto

```
proyecto/
│
├── index.html
├── css/
│   └── styles.css
│
├── js/
│   └── script.js
│
└── assets/
    └── img/
        ├── imagen1.jpg
        ├── imagen2.jpg
        └── logo.png
```

---

## Cómo usar el proyecto

1. Clonar el repositorio

```
git clone https://github.com/usuario/proyecto.git
```

2. Entrar a la carpeta del proyecto

```
cd proyecto
```

3. Abrir el archivo `index.html` en el navegador.

## Características
